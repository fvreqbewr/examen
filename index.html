Pagina 66 y 104

Esta página web cuenta con el HTML y CSS proporcionados por el libro, y como se aprecia en la imagen, presenta un formulario diseñado para registrar inventarios del negocio "Ediciones Fares". A través de este formulario, es posible asignar un código, precio y fecha a cualquier artículo que se cargue mediante el botón "Seleccionar Archivo". El estilo visual fue configurado según las indicaciones del libro.

Pagina 72

Esta página web inicialmente solo incluía el HTML proporcionado por el libro, pero debido a algunos errores, tuve que agregar el CSS. Su funcionalidad consiste en registrar la información para obtener libros de "Ediciones Fares", e incluye casillas para seleccionar el curso y un botón para enviar los datos.

Pagina 222

Este formulario originalmente estaba basado en PHP y W3CSS, pero como no estamos trabajando con esas tecnologías, tuve que modificar el código para usar únicamente etiquetas HTML y CSS. El diseño aún no coincide completamente con lo mostrado en el libro, aunque es posible que realice ajustes más adelante.

Pagina 255

Esta página web presenta una galería de imágenes, donde el libro únicamente proporcionó una imagen a modo de guía, por lo que tuve que desarrollar tanto el HTML como el CSS. Las imágenes incluyen un efecto hover y un hipervínculo especial que permiten obtener una "vista previa". Al igual que en otros proyectos, decidí usar libros de "Ediciones Fares" como base para las imágenes.
